GenOpt example files for IDA

The files in this directory illustrate how to use GenOpt with IDA. They have been provided by Mika Vuolle and have been used for the example published by Hasan et al., 2008.

References

Hasan, Ala, and Mika Vuollea and Kai Sirén. Minimisation of life cycle cost of a detached house using combined simulation and optimisation. In press: Building and Environment, 2008. doi:10.1016/j.buildenv.2007.12.003.

For more information, visit the GenOpt web pages.
